# はじめに

先日、Anaconda の環境構築を行いました。私は研究を始めるまで Python にあまり触れてこなかったため、環境構築の際に`pip`やら`pip3`やら`pyenv`やら`conda`やらで訳分からなくなっていました。
しかし Anaconda を用いて環境構築することで、 Python のエコシステム周りがなんとなーく分かってきました。これらを忘れないうちにまとめておきます。
